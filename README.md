# Technical notes | Marceli Grabowski

Static Astro site for technical notes about complex backend, cloud and legacy systems.

Public site: <https://marceligrabowski.github.io/>

## Local development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Content

Published notes live in `src/content/notes/`.

This site is intentionally separate from the professional homepage at <https://www.marceligrabowski.pl/>, but shares the same visual identity and links back to it.
